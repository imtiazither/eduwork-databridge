# ADR 0002: Core Technology Stack

Status: Accepted

Use Python 3.12, FastAPI, SQLAlchemy 2, Alembic, PostgreSQL, Pydantic v2, React/TypeScript, Vite, and Docker Compose. Later data phases add Polars, DuckDB, Pandera, RapidFuzz, Splink, Dagster, OpenLineage, and OpenTelemetry behind optional extras.
