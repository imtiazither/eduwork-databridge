# Claims Register

| Claim | Allowed now? | Required evidence |
|---|---:|---|
| Phases 0–2 implemented | Yes, after acceptance checks pass | Source, tests, migration, generated docs, clean-clone verification |
| Production data integration complete | No | Later connector, ingestion, mapping, validation, and operational tests |
| Used by a company | No | First-hand company evidence tied to actual work |
| Improved learning or workforce outcomes | No | Appropriate real-world evaluation |
| Certified or legally compliant | No | Formal conformance or compliance review specific to deployment |
